# fcop 3.2.5 — Rule 0.a 协作闭环 · Hot/Cold Path · parent API

**发布日期**：2026-05-30  
**包版本**：`fcop==3.2.5` · `fcop-mcp==3.2.5`（双包 lockstep，per [ADR-0002](../../adr/ADR-0002-package-split-and-migration.md)）  
**PyPI**：待发布

---

## 一句话

**协作闭环语义升级**：执行者 `report` 后停步，归档需 ADMIN/leader 授权；bundled 规则升至 **3.2.5**；34 份角色 charter 注入 Common Block；`write_task` 新增 **`parent`** 规范字段支持。

---

## 背景

| 议题 | 3.2.4 及以前 | 3.2.5 |
|------|-------------|-------|
| Rule 0.a.1 | 四步含执行者必 `archive` | `task → 执行/派发 → report → 停步`；archive 需授权 |
| 角色 charter | 各模板措辞漂移 | `_COMMON-FCOP-3.2.5.md` 统一注入 |
| `parent:` 字段 | 仅手工 frontmatter / extra | `TaskFrontmatter.parent` + MCP/Python API |
| MCP docstring | 部分仍暗示自动 archive | 14 工具对齐 Hot/Cold Path（工具总数仍 45） |

---

## 变更摘要

### Changed — `fcop`

- **Rule 0.a.1 重写** + **Rule 0.a.2–0.a.6**（Hot/Cold Path、生命周期≠业务完成、主子任务、授权归档、REPORT 停步）。
- **Bundled rules** — `fcop_rules_version` / `fcop_protocol_version` → **3.2.5**。
- **`write_task(..., parent=...)`** — 派生链路上游任务 ID 可程序化写入 `parent:` frontmatter。
- **`scripts/inject_workflow_constraint.py`** — 34 份 `roles/*.md` 注入 Common Block。
- **`letter-to-admin.{zh,en}.md`** — v3.2.5 顶部摘要。

### Changed — `fcop-mcp`

- **14 个工具 docstring**（3.2.5 约束范围，不扩至全部 45 工具）。
- **`write_task` / `create_task`** — 新增 `parent` 参数。

### Docs / Tests

- `docs/releases/3.2.5.md`（本文件）、`CHANGELOG.md` [3.2.5] 条目。
- `TestMcpToolDocstrings325` 14 passed；`test_role_templates` / `test_rules_text_regression` 更新。

## 升级指引

```bash
pip install -U "fcop>=3.2.5,<3.3" "fcop-mcp>=3.2.5,<3.3"
```

在 IDE 中由 **ADMIN** 执行 MCP 工具 **`redeploy_rules()`**（或 Python：`Project.deploy_protocol_rules(force=True)`），刷新项目根四份规则文件。

从 3.2.4 升级：无破坏性 API 移除；若项目角色文档仍含旧「四步必 archive」表述，可运行 `deploy_role_templates(force=True)` 或重跑 `inject_workflow_constraint.py` 后审 diff。

## English summary

**fcop 3.2.5** upgrades the **collaboration cycle** (Rule 0.a): executors stop after REPORT; archive requires ADMIN/leader authorisation. Bundled rules bump to **3.2.5**. **34 role charters** get a shared Common Block. **`write_task(..., parent=...)`** makes `parent:` a first-class frontmatter field. **14 MCP tool docstrings** align with Hot/Cold Path semantics (45 tools total unchanged).
