## What's new vs 0.7.2

详细每一段见 [`CHANGELOG.md` §1.0.0-rc.1](../../CHANGELOG.md)。
本节只列**用户可见的 API 增量**：

### 1. 4 类 IPC envelope（新增 REVIEW）

```python
from fcop import Project, ReviewDecision

project = Project(".")
review = project.write_review(
    reviewer_role="ME",
    subject_type="task",
    subject_ref="docs/agents/tasks/TASK-20260509-001-ADMIN-to-ME.md",
    decision=ReviewDecision.APPROVED,
    rationale="implementation matches charter",
)
```

### 2. Failure & Recovery API

```python
from fcop import Failure, FailureType, RecoveryAction

f = Failure(
    failure_type=FailureType.TIMEOUT,
    subject_agent_code="ME",
    detected_at=datetime.now(timezone.utc),
    subject_task_id="TASK-20260509-007",
    suggested_recovery=RecoveryAction.ABORT,
)

project.report_failure(f)
outcome = project.apply_recovery(f)  # ABORT 写一份 status=aborted REPORT
```

### 3. Event subscription

```python
def on_event(ev):
    print(ev.event_type, ev.subject)

sub = project.subscribe_events(callback=on_event)

# Trigger file-derived events:
project.poll_once()  # 显式调；v1.0 不引入后台线程

# Trigger in-process events:
project.report_failure(f)        # → FAILURE_DETECTED
project.apply_recovery(f)        # → RECOVERY_INITIATED + RECOVERY_COMPLETED

sub.unsubscribe()
```

### 4. Boundary 强制

```python
# Worker layer 默认不能 review governance subject
project.assert_boundary("DEV", "review_decision", target_role="LEADER")
# → BoundaryViolationError + 自动 emit BOUNDARY_VIOLATED 事件
```

### 5. JSON Schema 校验

```python
from fcop.core.jsonschema_validator import validate_envelope_frontmatter

issues = validate_envelope_frontmatter(frontmatter_dict, "TASK")
for issue in issues:
    print(issue.severity, issue.field, issue.message)
```
